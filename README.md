This is a simple To-Do List web application built using **Flask (Python)** and **HTML/CSS**.  
The app allows users to add tasks with a due date, display them on the page, and mark them as completed.  
If a task's due date has passed, the task will be shown as **Overdue**.

---

## Features
- Add tasks with a due date
- Display tasks in a clean and aesthetic interface
- Automatically show whether a task is overdue or still due
- Remove tasks by checking the checkbox
- Current date is shown at the top of the page
- Motivational message *‚ÄúKeep Going Forward!‚Äù* for encouragement

## How to Run the Project
1. Make sure Python is installed on your device.
2. Install Flask if not installed yet:
   pip install flask
3. Run the Python app:
   python app.py
4. Open your browser and go to:
   http://127.0.0.1:5000
